# 食物不耐受记录网站

一个简单易用的本地食物不耐受记录工具，帮助您追踪每日的食物摄入情况。

## 📋 包含的25种常见不耐受食物

### 🥛 奶制品
- 牛奶、奶酪、酸奶

### 🌾 谷物类
- 小麦、大麦、黑麦、燕麦、大米

### 🥚 蛋类
- 鸡蛋

### 🥜 坚果类
- 花生、杏仁、核桃

### 🦐 海鲜类
- 虾、蟹、鱼

### 🫘 豆类
- 黄豆、花生

### 🥬 蔬菜类
- 芹菜、生菜、番茄、洋葱、土豆

### 🍌 水果类
- 香蕉、橙子、草莓

### ☕ 其他
- 咖啡、巧克力、蜂蜜

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击日期两侧的箭头切换日期，或点击"今天"按钮快速回到今天
3. 点击食物卡片即可记录当日是否食用了该食物（红色表示已食用）
4. 数据自动保存在浏览器本地存储中

## ✨ 功能特点

- 📱 响应式设计，支持手机和电脑
- 💾 本地存储，数据不会丢失
- 📊 统计功能，查看每日摄入情况
- 📅 历史记录，随时回顾过往饮食
- 🎨 美观的卡片式界面
